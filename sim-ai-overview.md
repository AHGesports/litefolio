# Sim AI - Overview

## What is Sim AI?

Sim is an **open-source visual workflow builder** for designing, deploying, and monitoring AI agent workflows. It enables users to create intelligent automation systems using a no-code interface that connects AI models, databases, APIs, and business tools through a drag-and-drop canvas.

Used by **60,000+ developers**, Sim provides enterprise-grade security with SOC2 & HIPAA compliance.

## Core Features

| Feature | Description |
|---------|-------------|
| **Visual Canvas** | Drag-and-drop blocks to build AI workflows without coding |
| **Modular Block System** | Specialized components for processing, logic, and output tasks |
| **Copilot Assistant** | AI-powered suggestions and explanations in real-time |
| **Knowledgebase** | Processes documents into searchable chunks with vector search |
| **Real-Time Collaboration** | Teams can edit workflows simultaneously with live updates |
| **Flexible Execution** | Launch via chat, REST APIs, webhooks, scheduled cron jobs, or external events |
